About me

Who I am

I'm Michał, an 18-year-old frontend developer based in Poland. I started programming out of curiosity and quickly fell in love with building things that run in the browser.

My main stack is Vue.js with TypeScript — I know how to structure real applications using Vuex for state management, Vue Router for navigation, and Vite for a fast dev experience. I also work comfortably with Node.js + Express for backend APIs and Webpack for bundling custom setups.

I learn by building — not just tutorials. The projects below are real, working applications I built from scratch, including a full e-commerce store and a browser game with collision detection, scoring and animations.
